== Allgemeine Informationen
Ein Domänenmodell ist die grafische Darstellung von Beziehungen zwischen den Entitäten mit zugehörigen Attributen.

In diesem konkreten Fall beschreibt das Domänenmodell grafisch die Funktionsweise und die Abhängigkeiten innerhalb der Software für die Ressourcenverwaltung.

Das Ziel ist zu zeigen, dass Benutzer andere Benutzerkonten anlegen und Projekte erstellen können. Die angelegten Benutzer werden im Kalender angezeigt. Projekte können an Benutzer pro Tag durch einen Status vergeben werden. Darüber hinaus können Stati anderen Benutzern die Abwesendheit eines anderen Benutzers signalisieren. Außerdem ist es wichtig, dass Feiertage in das Kalender zu implementiert werden. Damit wird Übersichtlichkeit und bessere Wochenplanbarkeit garantiert.
